Circuit is a blockchain accelerator programme run by NewCampus in partnership with Base (Coinbase) and Soul World Bank, targeting Southeast Asian Web3 founders with live products who are seeking to raise capital and launch token generation events.
This repository hosts two investor-facing tools built for Cohort 1:
	•	Investor Data Room — a founder profile browser where pre-qualified investors can review cohort snapshots, filter by sector, and express interest before the programme begins
	•	Investor Application Form — a 6-step onboarding form for investors to declare their thesis, sector focus, cheque size, and availability for Friday Singapore meeting sessions
